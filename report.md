##Region Chosen

I selected the UK South Azure region because it is geographically closer to Nigeria, which helps reduce network latency and improve application performance.

##Shared Responsibility Model

For the Storage Account:

Microsoft Azure is responsible for the physical infrastructure, networking, hardware maintenance, and availability of the storage service. I am responsible for securing access to the storage account, managing permissions, protecting data, and configuring the resource correctly.
